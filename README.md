# Recivu

> Trasformiamo gli scontrini fiscali in fatture elettroniche, recuperando l'IVA per le aziende.
> We turn fiscal receipts into electronic invoices, recovering VAT for companies.

---

## 🇮🇹 Italiano

### Cos'è Recivu

Recivu è una piattaforma B2B che consente alle aziende di recuperare l'IVA sulle spese aziendali, convertendo automaticamente gli scontrini fiscali in fatture elettroniche. Il processo avviene negoziando la fattura direttamente con il commerciante, senza richiedere alcuna azione manuale da parte dell'azienda cliente.

Recivu si integra con le principali piattaforme di gestione delle spese tramite API, permettendo un flusso completamente automatizzato dalla nota spese alla fattura elettronica.

### API

Le API di Recivu consentono ai partner autorizzati di interagire con l'ecosistema Recivu in modo programmatico. Le funzionalità esposte coprono tre aree principali:

- **Scontrini** — Invia uno scontrino per avviare la conversione in fattura elettronica, monitora lo stato della richiesta e scarica il PDF della fattura emessa.
- **Aziende** — Registra e gestisci le aziende clienti all'interno dell'ecosistema Recivu.
- **Dipendenti** — Aggiungi e gestisci i dipendenti associati a ciascuna azienda, necessari per l'attribuzione delle spese.

> ⚠️ **L'accesso alle API è riservato esclusivamente ai partner autorizzati.** Per informazioni su come diventare partner, contattaci all'indirizzo indicato nella sezione Contatti.

## 🇬🇧 English

### What is Recivu

Recivu is a B2B platform that enables companies to recover VAT on business expenses by automatically converting fiscal receipts into electronic invoices. This is achieved by negotiating the invoice directly with the merchant, requiring no manual action from the client company.

Recivu integrates with leading expense management platforms via API, enabling a fully automated flow from expense report to electronic invoice.

### API

Recivu's API allows authorized partners to interact with the Recivu ecosystem programmatically. The exposed functionality covers three main areas:

- **Receipts** — Submit a receipt to initiate conversion into an electronic invoice, track the status of the request, and download the PDF of the issued invoice.
- **Companies** — Register and manage client companies within the Recivu ecosystem.
- **Employees** — Add and manage employees associated with each company, required for expense attribution.

> ⚠️ **API access is reserved exclusively for authorized partners.** For information on becoming a partner, please reach out via the contact details in the section below.
